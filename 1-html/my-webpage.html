Lists

ul tags create unordered lists. This in particular is a bullet list.

<ul>
  • <li>list item 1 </li>
  • <li>list item 2 </li>
  • <li>list item 3 </li>
</ul>

ol tags create ordered lists. This in particular is a numbered list.

<ol>
  1. <li>list item 1</li>
  2. <li>list item 2</li>
  3. <li>list item 3</li>
</ol>

You can change the starting value by using:

<ol start = "10">
  1. <li>list item 1</li>
  2. <li>list item 2</li>
  3. <li>list item 3</li>
</ol>

Or you can also reverse the list by using:

<ol reversed>
  1. <li>list item 1</li>
  2. <li>list item 2</li>
  3. <li>list item 3</li>
</ol>

Ordered lists have different types. To explore these different types you use the following syntax and select what you need.

<ol type = "A">
  1. <li>list item 1</li>
  2. <li>list item 2</li>
  3. <li>list item 3</li>
</ol>

Images

Insert images using <img src = "file_name.jpg"/>. file_name.jpg can be replaced with a url link as well; this is called hotlinking.

Image proportions can also be altered by adjusting variables like width, height, and align.
<img src = "file_name.png" width = "100" height = "100" align = "left"/>
